# TechFlow Design System

## Token Format
Stitch (exported)

## Brand Identity
- IT 컨설팅 전문 기업의 신뢰감과 전문성을 전달
- 깔끔하고 모던한 비즈니스 톤
- Deep navy blue를 primary로 사용하여 안정감과 전문성 표현

## Design Principles
- Clean, professional, trustworthy
- Generous whitespace for readability
- Strong visual hierarchy with clear CTAs
- Corporate yet approachable feel
- Subtle gradients and shadows for depth

## Design Language
- Brand personality: formal / professional
- Visual density: spacious
- Color philosophy: analogous (navy → blue → sky blue)
- Color variant: TONAL_SPOT

## Platform Strategy
- Target platforms: Web (Desktop)
- Responsive approach: Mobile-first

## Typography
- Font family: Inter (headline, body, label)
- Type scale: See tokens.json typography section

## Color Palette
- Primary (Deep Navy `#1E3A5F`): Headers, CTAs, key elements
- Secondary (Blue `#2563EB`): Interactive elements, links
- Accent (Sky Blue `#0EA5E9`): Accents, highlights
- Neutral (Slate `#64748B`): Body text, borders
- White backgrounds with subtle gray sections for contrast
- Semantic: success `#34C759`, warning `#FF9500`, error `#FF3B30`

## Component Style
- Cards: White background, subtle shadow, rounded-lg
- Buttons: Solid primary with hover state, ghost variant for secondary actions
- Forms: Clean borders, focus ring in primary color
- Icons: Line style, consistent 24px size
- Naming convention: PascalCase components, camelCase props
- State coverage: hover, focus, active, disabled, loading, error, empty

## Spacing System
- Base unit: 4pt grid (spacingScale: 2)
- Scale: xs(4), sm(8), md(16), lg(24), xl(32), 2xl(48), 3xl(64)

## Border Radius (ROUND_EIGHT)
- sm: 4px, md: 8px, lg: 12px, xl: 16px

## Responsive Breakpoints
### Web
- sm: 640px, md: 768px, lg: 1024px, xl: 1280px
